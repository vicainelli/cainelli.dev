## Editor + Terminal

I started using Zed editor in the end of 2023, and I've been enjoying it ever since.
The only thing I miss is the integration with Wallaby/Quokka.

- IDE: [Zed](https://zed.dev/?ref=cainelli.dev)
- Terminal: [Ghostty](https://ghostty.org/?ref=cainelli.dev)

### Zed extensions

Since I started using Zed, I noticed I need less extensions than I thought. Here are the ones I'm currently using:

- Astro
- Emmet
- HTML
- Material Icon Theme
- Context7 MCP Server
- Dracula Light (Alucard)
- Nord Dark
