When I joined Just Eat Takeaway in 2020, squads were rebuilding similar UI components in different ways. That slowed everyone down and made the product feel inconsistent.

I helped build a Vue design system that ended up supporting around 10 squads. The hard part was not the components themselves; it was the system around them. I defined API conventions, versioning rules, and migration guides so teams could adopt changes without breaking product delivery.

Governance mattered. We wanted teams to move fast without turning the design system into a free-for-all. I wrote contribution guidelines, added Jest checks in CI, and made sure API changes were deliberate.

I also mentored teams on component architecture and testing habits. The payoff was smoother onboarding and fewer regressions during releases.
