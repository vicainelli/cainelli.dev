I have been testing [Claude Code](https://claude.ai/code) with [z.ai's GLM 4.7](https://z.ai/models/GLM-4.7).

For my workflow, it handles tool calls reliably and keeps costs lower than running everything through premium models.

If you use Claude Code often and want a cheaper model for routine tasks, this combo is worth a try.
