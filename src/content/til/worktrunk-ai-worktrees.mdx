I have been using [WorkTrunk](https://worktrunk.dev/) and it has been really useful for AI-assisted coding.

When an agent is generating code, I often want to try ideas in different branches without messing up my main working copy. WorkTrunk makes that flow feel simple: each branch can live in its own Git worktree, so I can switch context, compare attempts, and clean things up without the usual headaches.

If you use AI coding tools and keep juggling branches, WorkTrunk is worth trying.
